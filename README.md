# Data-Structure
 
A collection of data structures implemented in C++, built for learning 
and practicing core computer science concepts.
 
## 📌 About
 
This repository includes implementations of fundamental data structures, written from scratch in C++ 
to better understand how they work internally.
 
## 📂 Contents
 
- Arrays & Linked Lists (Singly / Doubly)
- Stacks & Queues
- Trees (Binary Trees, BST)
- Hash Tables
- (More structures added as I learn them)

## 🛠️ Tech Stack
 
- **Language :** C++
## 🚀 How to Run
 
```bash
g++ filename.cpp -o output
./output
```
